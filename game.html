game

This game’s goal is to teach you the proper cat petting etiquette by rewarding the player when they hover their mouse in a good zone by loading a little bit of a progress bar and detracting a little bit of the progress bar when they hover over a bad area to pet a cat. Some areas will even automatically result in a game over… because we all know cat’s don’t like belly rubs

function checktail(){
  if (disttail < tail){
 test = 1;
 tailprog++;
 if (tailprog < 100){
 prog = prog +1/2;
 }
}
}
